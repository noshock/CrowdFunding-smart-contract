# Crowdfunding Smart Contract

A decentralized Crowdfunding Smart Contract built using Solidity that enables users to contribute ETH to a fundraising campaign in a secure and transparent manner.

The campaign creator can withdraw funds only if the fundraising goal is achieved after the campaign deadline.

 ## Overview

Traditional crowdfunding platforms depend on centralized intermediaries. This smart contract demonstrates how crowdfunding can work on blockchain using Ethereum and Solidity.

The contract allows:

A creator to start a fundraising campaign
Users to donate ETH
Automatic deadline management
Goal-based withdrawal system
Secure ETH transfer

## Features

 Create crowdfunding campaign
 Set funding goal
 Set campaign deadline
 Donate ETH to campaign
 Track total funds raised
 Store donor contributions
 Creator-only withdrawal
 Goal-based withdrawal validation
 Secure ETH transfer using call()
